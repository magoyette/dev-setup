# Agent Context

The Available CLI Tools list contains CLI tools that are installed and can be used as alternatives to the Linux tools installed on all machines.

Check `--help` for exact flags when needed.

## Available CLI Tools

- `rg`: fast recursive search. Prefer over `grep` for file and content search.
- `jq`: parse and edit JSON.
- `shellcheck`: lint shell scripts, should be used after a `.sh` file has been created or modified.
- `difft`: syntax-aware diff.
- `fzf`: interactive fuzzy finder.
- `pandoc`: convert Markdown and other document formats.
