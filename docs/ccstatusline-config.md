# ccstatusline config

- In Powerline Setup:
  - Install Powerline Fonts then restart
  - Enable Powerline Mode
  - Align Widgets: Enabled
  - Set Theme to "One Dark"
  - Set Powerline Separator to "Round Right"
  - Set Powerline Start Cap to "Round"
  - Set Powerline End Cap to "Round"
- In Terminal Options:
  - Set Terminal width to "Full width until compact" at 60%
  - Set Color Level to "Truecolor"
- Line 1:
  1- Model (raw value)
  2- Context % (usable) (raw value)
  3- Block Timer (raw value)
  4- Git Branch
  5- Git Changes
- Install to Claude Code
- Save & Exit
